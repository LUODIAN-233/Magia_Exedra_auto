# Magia Exedra 自動周回ボット

画像認識ベースの **Magia Exedra**（圆哆啦） Windows 向けゲーム自動化スクリプト

Link Raid と Crystalis の両方の自動周回モードをサポート。言語／解像度切り替えと素材の自動スケーリングを搭載

## 機能一覧

### Link Raid 自動周回

> 開始画面：ゲームのメイン画面（灯台画面）

- `backup_requests` 画面に自動で入り、指定レベルのチームを探して更新
- 目標レベルが見つからない場合はリストを自動で下へスクロール（最大 4 回）
- 終了したバトル（joined battles）を自動で片付け
- バトル終了後に自動で「いいね」を送信
- 体力薬の使用回数を設定可能（**0〜10 回**）
- LV6〜LV12 のレベル選択に対応

### Crystalis 自動周回

> 開始画面：編成完了後の画面（`play` を押すと戦闘に入る画面）

- ステージを自動で繰り返し周回し、各バトル終了後に `retry` をクリックして継続
- 体力薬の使用回数を設定可能（**0〜8 回**）
- 体力が尽き、かつ薬の回数を使い切ると自動停止

### 共通機能

- 起動時にゲームウィンドウを最前面に表示
- 停止ボタンでいつでも任意の周回タスクを中断可能
- 言語／解像度切り替え（英語・日本語に対応、複数解像度の素材を自動スケーリング）
- 720p のオリジナル素材を 1080p／1440p／4K にワンクリックでスケーリング

## 仕組み

| モジュール | 技術手法 |
|:-----:|:---------|
| 画像認識 | OpenCV `TM_SQDIFF_NORMED`、マッチングしきい値 `0.8` |
| クリック操作 | PyAutoGUI（移動してからクリック。そうしないとゲームが入力を無効と判定） |

| 素材スケーリング | ImageMagick（`tools/ImageMagick/magick.exe`） |
| テンプレート切り替え | Windows のジャンクション（junction）。`aim/` は `language/` 配下の実パックを指す |

## 操作説明

### 1. Link Raid 周回

ゲームのメイン画面（灯台画面）で起動する必要があります

- 周回するレベルを選択（LV6〜LV12）
- 体力薬を飲む回数を設定（0〜10）
- 「**link raid挂机启动**」をクリック（Link Raid の開始ボタン）

### 2. Crystalis 周回

編成完了後の画面で起動する必要があります（`play` を押すと戦闘に入る画面）

- 体力薬を飲む回数を設定（0〜8）
- 「**自动刷晶花，需要在play界面启动**」をクリック（Crystalis の開始ボタン）

### 3. 言語／解像度切り替え

- ドロップダウンで言語と解像度を選ぶと**自動的に切り替わる**（追加の確定ボタン不要）
- 言語ドロップダウンは中国語名（英語／日本語）で表示、データ部は実際のコード
- 空のテンプレートパックは「（空）」と表示され、切り替えは拒否される

### 4. リストを更新

720p のオリジナル素材を他の解像度へ自動スケーリングします：

| 元の解像度 | → スケール先 | 倍率 |
|:--------:|:----------:|:----:|
| 1280×720 | 1920×1080 | 1.5x |
| 1280×720 | 2560×1440 | 2x |
| 1280×720 | 3840×2160 | 3x |

> 既存のターゲットファイルはスキップされます。再生成したい場合は該当ファイルを削除してください。

## 注意事項

- **ゲームウィンドウは最前面にある必要があります**。周回ソフトはバックグラウンドでも動作可能（他のウィンドウで隠れても影響なし）
- Link Raid の初回クリックは**ハードコード座標**（`1000,500` → `1200,600`）を使用しており、1280×720 でのみ正常動作
- 解像度切り替え後、Link Raid の初回クリックは手動で調整が必要（右下をクリック）
- ゲームは**16:9**のウィンドウモードで実行し、テンプレートの言語をゲームの言語と一致させる必要があります
- JP サーバーはゲーム内で EN に切り替えることで、英語テンプレートを再利用でき、スクリーンショットを撮り直す必要はありません

├── resource/
│   └── main.ico             # プログラムアイコン

### コア規約

- **戻り値規約**：`2` = 成功／発見／クリック済み、`1` = 未発見／継続リトライ。ブール値ではない。
- **テンプレート番号**：`click_item_with_result` は `<name>_1.png`、`<name>_2.png`… とファイルが存在しなくなるまで順に試行。変体を追加するには次の番号の画像を置くだけ。
- **停止フラグ**：`guaji_1`／`guaji_2` はグローバル変数でロックなし。`stop_any` スレッドが書き換え、ワーカースレッドが読み取る。

## 補足説明

このスクリプトは**スマートフォンでは実行できません**（ADB 経由での制御不可）。ゲームは ADB 接続を検出すると実行を拒否し、アンチチート機構がキーボード入力スクリプトを制限します。ウィンドウの画像認識に基づく PC 方式が、現在のところハードルが最も低い妥協案です。
