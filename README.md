运行原理是图像识别的点击。
点击部分：PyAutoGUI
窗体部分：Pyside6
最新提示：根据b站评论区反馈，日服改成英语能够直接使用，那就不需要再截图一次了，这个非常痛苦的

实现功能：link raid挂机和晶花挂机
  link raid挂机可以实现尝试寻找对应要挂的级别，喝体力药，清理打完的对局，打完点赞。会自动下滑增加软件效率。
  晶花部分可以实现打完体力之后喝药
  任意挂机启动后，会自动把游戏拉到前台，无需手动拉到前台。

如何启动：
  1.完成需要的设置
  2.link raid挂机需要在游戏主界面启动（灯台界面）。
    晶花挂机需要在选中队伍界面启动，即为点击play后就开始游戏的界面。
  
使用提示：
  需要使用1280*720分辨率，窗口化运行游戏，画质任意，我测试下来不影响。
  游戏界面必须在最前面，并且你不能同时干别的，挂机软件不需要在前台，挂机软件被其他窗口挡住了也问题不大。
  注意：如果使用了其他分辨率的截图，那么第一次点击的位置会失效，需要手动点击右下（只会影响link raid）
  
关于其他补充
  这个游戏不能做成手机挂机的软件，即为类似明日方舟挂机。原理是使用ADB截图并点击。因为这游戏你打开ADB就不让运行了，别和我说什么ROOT之后隐藏软件检测ADB啥的，软件门槛过高会导致绝大多数用户无法使用，另外这游戏反作弊及其恶心，会限制脚本使用键盘进行控制。

关于其他语言和其他分辨率：
  我自己目前完成了1280*720的截图，切换语言请把文件夹重命名成aim，重命名前记得把原来的改名字。不要把两个文件夹的内容叠加到一起，否则会极大影响效率或者出bug。但是日语的截图我后面就没有管过，目前无法使用他人贡献的也同理。
  因为软件本质只是识别图像并点击，如有需求可以自行截图实现其他分辨率和其他语言。也可以尝试使用分辨率转换小工具自行转换（会影响稳定性）。
